## Changelog

### 0.1.0

- Initial release of `Mindstash` with:
  - Sidebar view for browsing, searching, editing, and deleting ideas
  - Quick capture popup with markdown-style formatting support
  - Workspace-scoped JSON storage for ideas
- Updated default keybindings:
  - `Ctrl+Shift+I` / `Cmd+Shift+I` – Open Mindstash sidebar
  - `Ctrl+Shift+A` / `Cmd+Shift+A` – Open capture popup
  - `Ctrl+Shift+R` / `Cmd+Shift+R` – Append idea via quick input
